在线旅游（携程、途牛、同程、去啊）竞品分析

Apr 16, 2016 | 分析 | Hits

近年来，在资本大量涌入在线旅游业，在线旅游企业实现了市场的细分与整合，中国在线旅游市场已经从过去的探索期与市场启动期过渡到了市场的高速发展时期，行业竞争也已经到了白热化阶段。本文尝试对行业内知名在线旅游业产品进行分析。

二、市场概况

总体情况

2月4日，中国领先的旅游市场研究咨询机构劲旅咨询最新发布《2015-2016年中国在线旅行社市场研究报告》，报告显示，2015年，中国旅游市场总交易规模约为41300亿元，其中在线市场交易规模约达5402.9亿元，较2014年3670亿元同比增长47.2%，在线渗透率为13.1%，较2014年11.3%增长了约2个百分点。

其中各细分领域市场，在线机票、在线住宿和在线旅行社预订市场规模分别约为3431.5亿元、1141亿元和735.5亿元，占比分别约为63.5%、21.1%和13.6%。相较2014年，在线机票和在线旅行社预订交易额占比微幅增长，而在线住宿预订交易额占比略降。

在线旅游市场份额

携程旅行网和去哪儿网的交易额占比分别为36.8%和28.7%，阿里旅行与同程旅游的占比分别约为6.5%和5.1%，美团酒店+旅游、艺龙旅行网和途牛旅游网三家的占比分别为3.1%、2.5%、2.4%。

小结

在整个旅游行业中，在线旅游渗透率虽然还不高，但是渗透率仍然会持续增加；携程、去哪儿和艺龙三家的市场份额占到了整个在线旅行市场份额的七成，阿里旅行在从淘宝中分拆出来后依托阿里的大平台成为在线旅游行业的后起之秀。因为携程、艺龙、去哪儿因为合并为携程系所以在本次分析中艺龙和去哪儿不归为竞品，故选取主营业务是在线旅游的携程、阿里、途牛、同程为本次分析的主体。

小结

  1. 四款应用的大部分用户主要集中在北上广等沿海地区，比较有特点的是内陆地区中，四川地区的用户使用在线旅行的状态较其他内陆地区明显；
  2. 携程、途牛、同程的基本上一半的用户群体集中在30-39岁之间，该年龄段用户大部分可能是带孩子的家庭旅游，有时间也有心情，比较有特点的是阿里旅行，20-29岁之间的用户群体超过30岁以上用户，说明阿里旅行的目标用户会更加年轻态。同时，四款应用的用户群体大部介于20-40岁之间，该年龄段用户有精力也有物质基础；
  3. 携程和途牛的用户男女比例基本平衡，而同程和阿里用户男性用户明显多于女性用户，尤其阿里旅行，男性用户的比例基本上达到90％，说明女性用户更多关注携程和途牛两款，基本很少使用阿里旅行。

四、战略层分析

携程旅行途牛旅游同程旅游阿里旅行
Slogan携程在手，说走就走让旅游变的更简单，要旅游，找途牛快乐每一程世界触手可及
产品定位中国领先的综合性旅行服务公司，提供集无线应用、酒店预订、机票预订、旅游度假、商旅管理及旅游资讯在内的全方位旅行服务途牛以“让旅游更简单”为使命，为消费者提供旅游产品预订服务，涵盖跟团、自助、自驾、邮轮、酒店、签证、景区门票以及公司旅游等。专注于周边游及景点门票服务为用户提供提供国内机票/国际机票/酒店客栈/景点门票/国内国际度假旅游/签证/旅游卡券/租车/邮轮等旅游产品的信息搜索、购买、售后服务的一站式解决方案。
优势服务规模化和资源规模化是携程旅行网的核心优势。采用代理模式，抽取佣金。体量大，覆盖面广。更丰富的旅游线路，以及能满足客户不同需求的个性化定制路线。采用从旅行社采购来的旅游产品重新组合和包装，自行定价销售，赚取差价的集成模式。目标用户定位于有一定消费能力的白领阶层，主做服务品质和产品定制。将视线聚焦于短线周边旅游，主做景区门票的倾销，把坚持以追求平价的快速消费者为目标用户群，不断争夺促销性旅游市场。依托阿里大平台的强大势能，阿里旅行将在技术、支付、信用、金融等层面给消费者提供强大的平台架构和保障。平台大，价格上有一定的优势。全面进军OTA。

小结

  1. 携程旅行作为行业市场份额最大方，功能范围覆盖量最为广泛，几乎囊括了所有国内外旅游行业服务；
  2. 在在线旅游服务中阿里旅行的范围也几乎和携程一样全面，同时具有一个特色的酒店信用住服务，这跟阿里大平台积累的信用消费挂钩；
  3. 途牛旅游和同程旅游具有特色的理财服务，同时同程还提供特色的白条和分期旅游功能，因为同程的定位较其他平台瞄准低消费群体；
  4. 携程除了旅游服务外还提供电商功能－全球购，吸引海淘客。同阿里旅行，基于阿里平台，自然加入海外电商能力；
  5. 携程和同程提供火车票私人定制服务这一亮点功能，可以在线选座等；
  6. 携程、途牛和同程在截图的细节中有一处让人眼前一亮的功能设计，系统截图完毕后应用内会自动跳出是否分享的提示；
  7. 阿里旅行在点开行程的时候会自动提示是否读取短信记录行程，是一个兴奋性功能。

小结

  1. 携程旅行和阿里旅行从信息架构上来看，突出“旅行”工具，主打便捷出行的目的，主页主要位置突出机票、酒店等预订服务；
  2. 途牛旅游和同程旅游从信息架构上来看，突出“旅游”工具，主打旅游和行程规划，主页主要位置突出旅游方式推荐和规划；
  3. 途牛旅游、同程旅游结构基本一致，说明两者在业务方向和重点比较一致；
  4. 携程旅行从信息结构上看，比其他三者更重视客服服务，这和携程旅行服务的产品定位一致；
  5. 阿里旅行在便捷出行的定位上也比较突出旅游推荐，“发现”的入口比携程更靠前，位于一级入口，而携程的“发现”板块位于日程管理一级入口内部。

七、框架层分析

主要分析应用的导航结构：

  1. 四款应用导航结构大体相似，均为tab导航，不同tab功能明确；

  2. 携程首页导航突出酒店、机票、旅游三大板块的核心功能，每个大板块旁边的小板块作为大板块的补充和快捷入口，导航清晰，用户一目了然；

  3. 途牛和同程旅游首页导航采用突出特色旅游类别的方式，而途牛旅游在首页突出三个概括性的大类别，框架更加明确；

  4. 途牛、同程、阿里旅行在首页导航中用大图标表示主要功能，小图标表示次要功能，阿里旅行中在其后紧接着的小图标加解释展示优惠区，不同布局，形式多样，具有较好的层级展示和导航能力，但相比携程略显杂乱。

八、表现层分析

  1. 携程旅行采用格子的方式和不同的色块展示分类，扁平化的风格，整体设计比较具有商务感（成熟感）；
  2. 途牛旅游设计较为卡通，当前分析的时期正好在结合跑男做活动，图标设计继续采用了卡通的风格；
  3. 阿里旅行和同程旅游的视觉风格基本类似，图标风格和配色走小清新的路线。

九、总结

  1. 在线旅游的市场份额虽然还不及旅游行业总交易额的两成，但是随着时间的推移，在线旅游会越来越多的渗透到行业中，会有越来越多的人接受在线预订、购买的新方式；
  2. 与其它互联网服务一样，在线旅游的用户地域分布主要位于北上广及沿海地区，中西部地区还不广泛，唯有四川地区可以赶得上东部的平均水平，建议可以增加中西部的宣传投放力度；
  3. 四款旅游产品的人群年龄主要集中在20-40岁之间，40岁以上年龄人群缺乏开发，40－50岁之间的人群具备消费能力也具有开发潜力；
  4. 阿里旅行可以通过增设“亲子游”或其他家庭游套餐，用以增加更多30-40年龄段之间的这一极具消费能力的用户群体，也可以更多的发掘女性用户；
  5. 携程旅游在做好旅游主要业务的同时也可以跟进其他旅游产品所开展的互联网金融例如理财、信用消费（白条）等业务，一是信用消费也可以结合到旅游消费，促进旅游消费，二是旅游产品本来是具有极大现金流的平台，已经促使用户绑定银行卡＋消费，开展方便，阿里旅行最然没有在其中开展互联网金融相关业务，但是背靠阿里大平台不缺少此类资源和能力，所以在竞争因素的考虑下，具有竞争力的平台中目前携程不具备；
  6. 携程在信息结构中可以看出，在旅游产品的推荐中相比其它平台略显含蓄，这与携程本身的“旅行服务”这种工具化的定位和从机票、酒店在线订的历史发展因素有关；
  7. 同程旅游虽然具备特色的白条、理财能力，但是在自身主营业务旅行产品中较其他平台竞争力较弱，目前尚无国际酒店、租车等业务；
  8. 四款平台在业务领域都较成熟，通过相互借鉴和学习，在视觉、交互和业务逻辑方面都比较成熟。
